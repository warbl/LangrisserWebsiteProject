Purpose of the Site

The purpose of this page is to create something akin to an online deckbuilder for Langrisser Mobile. Users = Langrisser players, table = characters (one player can have many characters). https://sites.google.com/view/wikigrisser/home does not seem to have this functionality, so it would be useful. Most of my web development/design experience comes from self studying through FreeCodeCamp. I studied HTML/CSS, Javascript, JSON, how to use API's. I've not used this knowledge in quite some time, so this class would be good to formally study the topics.
